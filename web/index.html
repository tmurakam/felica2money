2008/3/16 : Version 2.4 公開しました。CSV ファイルの読み込み機能が追加されています。

FeliCa2Money とは？

FeliCa2Moneyは、パソリを使って Edy などの電子マネーの履歴を読み取り、 電子明細(OFX形式ファイル)に変換するものです。これを使えば、電子マネーの履歴を ワンタッチで家計簿に記帳できます。

また、CSV ファイルからの取り込みにも対応しています。

対応家計簿ソフト

Money電子明細(OFXファイル)には多くの家計簿ソフトが対応しています。以下はその一部です。

  • 製品
    • Microsoft Money
    • マスターマネー
    • てきぱき家計簿マム
    • ミラクル家計簿
  • シェアウェア
    • 家計簿ひかる
    • Let's 家計簿
  • フリー
    • ゆう子の家計簿

対応電子マネー

FeliCa2Money は、以下の電子マネーに対応しています。

  • Edy
  • nanaco
  • Suica
  • ICOCA
  • PiTaPa
  • PASMO
  • TOICA
