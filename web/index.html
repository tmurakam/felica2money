FeliCa2Money

News : '08/2/23: Version 2.0 公開しました。設定項目が追加されています。

FeliCa2Money とは？

このツールは、PaSoRi を使って Edy などの電子マネーの履歴を読み取り、 Microsoft Money の電子明細(OFX形式ファイル)に変換するものです。

現時点では、以下の電子マネーに対応しています。

  • Edy
  • Suica
  • ICOCA
  • PiTaPa
  • PASMO
  • nanaco
ただし、テストしたのは Edy / Suica / nanaco だけです。テスタ求む。

必要環境

  • USB ポートを備えた Windows PC : たぶん Windows Vista, XP, 2000 あたりなら問題なく動作します
  • .NET Framework 2.0 以降
  • SONY 非接触 IC カードリーダー/ライター (PaSoRi)
  • PaSoRi 基本ソフトウェア
  • SFCPeep
Version 2.0 から、.NET Framework が必要になりました。Windows Update などでインストールしておいてください。

SFCPeep は IC SFCard Fan からダウンロードしてインストールしておいてください。(注意: SFCard Peeper ではなくて、SFCPeep のほうが必要です。お間違えないように）
