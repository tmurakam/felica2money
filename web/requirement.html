必要環境

  • Windows 2000/XP/Vista/7 (x86/x64)
  • .NET Framework 2.0 以降
  • SONY 非接触 IC カードリーダー/ライター(FeliCaポート)
    • PC内蔵タイプまたは USB外付けタイプ(パソリ/PaSoRi)
  • PaSoRi 基本ソフトウェア

パソリについては SONY の RC-S370を推奨します。

また、Version 2.0 から、.NET Framework が必要になりました。Windows Vista/7 には最初からインストールされていますが、Windows2000/XPには入っていませんので、Windows Update などでインストールしておいてください。
